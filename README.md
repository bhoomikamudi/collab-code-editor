# Real-Time Collaborative Code Editor with AI Assistance

A browser-based collaborative code editor where multiple users can edit the same document in real time and use AI assistance for code completion, explanation, and codebase-aware chat.

## Planned Features

- User registration and login
- Document creation and management
- Real-time collaborative editing with WebSockets
- Operational Transformation for conflict handling
- Cursor presence tracking
- AI code completion
- AI explanation for selected code
- Codebase-aware AI chat using RAG
- Document snapshots and version history

## Tech Stack

- Frontend: React, CodeMirror 6, Tailwind CSS
- Backend: Node.js, Express, WebSocket
- Database: PostgreSQL
- Real-time state/cache: Redis
- AI service: Python FastAPI
- Vector database: ChromaDB
- AI models: OpenAI API
- Infrastructure: Docker and Docker Compose
